- [문제 출처](https://www.acmicpc.net/problem/10798) 

## 1. 설계 (입력, 출력, 조건, 순서)

## 2. 풀이 (code)
```python
arr = input().split(' ')
max_len = 0
for i in range(len(arr)) :
    if max_len < len(arr[i]) :
        max_len = len(arr[i])
        
for i in range(len(arr)) :
    temp = arr[i]
    while len(temp) < max_len :
        temp += ' '
    arr[i] = temp
    
for j in range(max_len) :
    for i in range(5) :
        if arr[i][j] != ' ' :
            print(arr[i][j], end='')

```
